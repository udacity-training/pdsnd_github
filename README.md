### Bikeshare Analytics Tool


### Description
This tool provides an interface between the user and bikeshare data from three cities (New York, Washington & Chicago)
It enables the user to view basic statistics on bike usage.

### Files used
The files used are:
* chicago.csv
* washington.csv
* new_york_city.csv
* bikeshare_2.py
